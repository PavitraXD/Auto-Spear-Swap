⚔️ What it does

Press R (Configurable in keybinds) and the mod will:

1. Find the first Spear + Lunge enchantment in your hotbar
2. Swap to it, fire a left-click attack
3. Instantly swap back to your original slot

If no Lunge Spear is found, you get a quick action-bar message.

## 📋 Requirements

- Minecraft **1.21.11/26.1/26.2**
- [Fabric Loader](https://fabricmc.net/) 0.18.1+
- [Fabric API](https://modrinth.com/mod/fabric-api) 0.141.4+1.21.11

> Spear and Lunge are **vanilla** content added in 1.21.11 — no extra mods needed.

---

## ⚙️ Keybind

Default key is **R**. change it anytime in **Options → Controls → AutoSpear**.

---

## 📝 Notes

- Client-side only — works in singleplayer and on any server
- Only checks hotbar slots 1–9
